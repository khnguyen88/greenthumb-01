Welcome to the Greenthumb App!
This app has two main features: A dashboard section and an AI chat messenger section.

Dashboard

The dashboard section consists of multiple dashboard pages that displays sensor readings, from the IoT Gardening System, in tabular and graphical format. The sensor readings shown on the dashboards are collected from the entirety of one growing session of a plant.

  • Provides measurement values, unit, and captured dates
  • View on Line Graph or Data Table
  • Simple and Elegant
  • Reponsive

AI Chat Messenger

The AI Chat Messenger is a chat messenger that has utilized Microsoft's Semantic Kernel's Agent and Handoff Orchestration frameworks and Kernel Functions to pull data from Adafruit IO and local JSON files to provide users analytical or factual information about the IoT Gardening System.

  • General Project Information
  • Sensor Hardware Information
  • Sensor Data Information and Analysis
  • General Plant Information
